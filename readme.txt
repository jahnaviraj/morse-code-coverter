A text based python program which takes in input of string and prints morse code. 
Also has decryption of morse code to text.
Takes in argument of method as --m or --method = t2m or m2t 
Takes in argument of string to convert to morse as --t or --text = "Some string to convert"
Takes in argument of string of morse code to convert to text as --c or --code = "-. Morse code --."
Either one of the two argument strings and method is mandatory.
